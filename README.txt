
HÁVAMÁL DIARIO - ANDROID/PWA

Puedes usar esta app de varias maneras:

1. Abrir index.html directamente.
2. Subirla a GitHub Pages.
3. Instalarla en Android como PWA:
   - Abrir desde Chrome.
   - Pulsar "Añadir a pantalla principal".

Funciones:
- Estrofa diaria automática.
- Estilo manuscrito nórdico.
- Guardar favoritas.
- Lectura por voz.
- Funciona offline.

También se puede convertir fácilmente en APK usando:
- PWABuilder
- Capacitor
- Android Studio
